Local AI Brain Architecture

Microservices architecture: FastAPI Gateway + vLLM-MLX (LLM) + Whisper (STT) + Kokoro (TTS) on Apple Silicon (MLX)

Backend Services

  • • vLLM MLX (LLM) on :8001
  • • STT Server (Whisper) on :8002
  • • TTS Server (Kokoro) on :8003
  • • API Gateway (FastAPI) on :8000
  • • Models loaded 24/7, hot in memory

Security & Monitoring

  • • Bearer token on ALL endpoints
  • • Memory Guard (54GB default)
  • • 429 rejection if exceeded
  • • Prometheus /metrics endpoint
  • • loguru structured logging

Deployment

  • • macOS Apple Silicon (MLX)
  • • launchd service (com.localbrain)
  • • Auto-start on boot
  • • uv-based dependency management
  • • ruff lint + pytest tests
